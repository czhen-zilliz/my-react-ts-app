# my-react-ts-app

## dev

### run client

1. cd `react` directory and run `yarn`

2. run `yarn dev`

### run server

1. cd `react` directory and run `yarn`

2. run `yarn build` under `react` directory

3. cp `react/dist` directory to `express`

4. cd `express` directory and run `yarn`

5. run `yarn dev` under `express` directory
